## 1) TSF / Interval-based：最小但非常强的“区间手工特征”

**Time Series Forest (TSF)** 在每棵树里随机抽多个区间 ([s,e])，并对每个区间提 3 个特征： ([arXiv](https://arxiv.org/abs/1302.2277?utm_source=chatgpt.com "A Time Series Forest for Classification and Feature Extraction"))

- **Mean**：$\mu_{s:e}=\frac{1}{n}\sum_{t=s}^{e}x_t$
    
- **Standard deviation**：$\sigma_{s:e}=\sqrt{\frac{1}{n-1}\sum(x_t-\mu)^2}$
    
- **Slope**（线性拟合斜率）：对区间内 $t$ 与 $x_t$ 做最小二乘直线拟合得到斜率
    
> 对脉冲类信号的直觉：某些区间的均值/方差会“突然变大”，斜率能捕捉上升沿/下降沿。

## 3) tsfresh：海量特征库里“最常用、最像手工”的一批（建议你从这里挑 20–60 个）

tsfresh 支持的特征非常多；下面是工程里最常见、且对脉冲/突变类信号友好的子集（名称就是 tsfresh 的 feature calculator 名）。 ([tsfresh](https://tsfresh.readthedocs.io/en/latest/text/list_of_features.html?utm_source=chatgpt.com "Overview on extracted features - tsfresh - Read the Docs"))

### 3.1 时间域幅值/变化类

- **abs_energy**：(\sum x_t^2)
    
- **absolute_sum_of_changes**：(\sum|x_t-x_{t-1}|)
    
- **mean / median / variance / standard_deviation**
    
- **skewness / kurtosis**
    
- **mean_abs_change / mean_change**
    
- **minimum / maximum**
    
- **quantile(q)**（如 0.1/0.5/0.9）
    
- **number_peaks(n)**（局部峰数量）
    
- **count_above_mean / count_below_mean**
    
- **index_mass_quantile(q)**（“质量”累计到 q 的位置）
    

### 3.2 自相关/依赖结构

- **autocorrelation(lag)**
    
- **partial_autocorrelation(lag)**
    
- **c3(lag)**（三阶相关）
    
- **cid_ce(normalize=True/False)**（复杂度估计）
    

### 3.3 频域/谱特征（尤其适合 HFCT）

- **fft_coefficient(coeff=k, attr=real/imag/abs/angle)**
    
- **fourier_entropy**
    
- **spectral_centroid**
    
- **agg_linear_trend**（在频域/分段上的趋势汇总）
    

### 3.4 拟合/趋势/平稳性

- **linear_trend(attr=slope/intercept/rvalue/pvalue/stderr)**
    
- **augmented_dickey_fuller**（ADF 平稳性检验统计量）
    
- **ar_coefficient(k)**（AR 模型系数）
    

> tsfresh 的关键不是“全用”，而是：先大范围提，再在 **GroupKFold 的训练折内**做筛选（避免泄漏）。

## 4) hctsa：7700+ 特征的“类别清单 + 代表性例子”

hctsa 的核心贡献是把大量科学文献里的时间序列分析方法统一成可计算特征，并提供筛选可解释特征的流程。它是 catch22 的母库。 ([科学直通车](https://www.sciencedirect.com/science/article/pii/S2405471217304386?utm_source=chatgpt.com "hctsa: A Computational Framework for Automated Time ..."))

你可以把 hctsa 的特征按“手工特征大类”理解为：

- **分布/统计矩**：均值、方差、偏度、峰度、分位数、离群强度
    
- **相关性/记忆性**：ACF/PACF、相关长度、互信息、长程相关
    
- **频域**：PSD 分带能量、谱质心/带宽、谱熵、主频稳定性
    
- **复杂度/不规则度**：样本熵/近似熵、Lempel–Ziv 复杂度、预测残差
    
- **非线性动力学**：相空间重构特征、Lyapunov 指标类、吸引子几何
    
- **尺度/分形**：DFA、Hurst 指数、多尺度波动
    
- **模型拟合参数**：AR/ARMA、状态空间、分段线性等模型的参数与残差统计
    

> 你学习 hctsa 的最佳方式：别试图背“7700 个”，而是用它做“特征雷达”，最后收敛到一小组你能解释的特征（这也是论文的主张）。

## 5) Shapelet / Shapelet Transform：把“判别性子序列”变成显式特征

Shapelet 方法的“特征”非常明确：**到每个 shapelet 的最小距离**。 ([Computer Science and Engineering](https://www.cs.ucr.edu/~eamonn/shaplet.pdf?utm_source=chatgpt.com "Time Series Shapelets: A New Primitive for Data Mining"))

- 对每个 shapelet (S)（长度 (l)），对时间序列 (T) 的所有长度 (l) 子序列做 z-normalize 后，计算欧氏距离
    
- **特征 = 最小距离**：(d(T,S)=\min_{R\in W_{T,l}} \text{dist}(S,R)) ([UEA Digital Repository](https://ueaeprints.uea.ac.uk/40201/1/LinesKDD2012.pdf?utm_source=chatgpt.com "A Shapelet Transform for Time Series Classification"))
    
- 最终得到一个向量 ([d(T,S_1),\dots,d(T,S_K)])，再接 Logistic/树模型分类
    

> 解释输出：哪几个 shapelet 距离小（匹配到）导致判为 PD，并能定位回原始波形的具体片段。

## 6) WEASEL：词袋特征（局部模式“词频”）+ 统计筛选

WEASEL 的特征本质是：把滑窗片段离散成“词”（SFA 等），然后统计 **unigram / bigram 词频**，再用卡方筛选，最后用线性分类。 ([arXiv](https://arxiv.org/pdf/1701.07681?utm_source=chatgpt.com "Fast and Accurate Time Series Classification with WEASEL"))

特征清单（按论文描述）：

- **Unigrams**：单个词的频次（每个词对应某种局部形态/频域离散符号）
    
- **Bigrams**：相邻两个词的联合频次（引入局部上下文）
    
- **Multiple window lengths**：多种窗口长度产生多尺度词
    
- **Chi-squared feature selection**：过滤无区分力的词特征
    
- **Linear classifier input**：最终是一个高维稀疏的词频向量
    

> 对脉冲信号：很适合把“干扰脉冲 vs PD 脉冲”的局部形态差异编码成可解释的模式词。

## 7) ROCKET：随机卷积核特征（非深度学习，但很强）

ROCKET 对每个随机卷积核输出两类显式特征： ([François Petitjean](https://francois-petitjean.com/Research/Dempster2020-Rocket.pdf?utm_source=chatgpt.com "ROCKET: Exceptionally fast and accurate time series ..."))

- **Global max pooling (MAX)**：卷积响应的最大值
    
- **PPV (Proportion of Positive Values)**：卷积响应中 (>0) 的比例
    

> 你可以把它理解成：用大量随机“模板”去扫序列，然后用 MAX/PPV 把“是否出现过某种局部模式”变成数值特征。
